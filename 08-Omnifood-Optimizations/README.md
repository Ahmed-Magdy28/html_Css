# Omnifood

Omnifood is an AI-powered food subscription service that delivers healthy, 365-days-per-year meals tailored to your personal tastes and nutritional needs.

## Features

- **Smart Meal Planning:** AI creates a 100% personalized weekly meal plan.
- **Diet Diversity:** Works with any diet including Vegan, Keto, Paleo, Gluten-free, and more.
- **Beautiful UI:** Modern, clean, and engaging design.
- **Fully Responsive:** Beautifully optimized across Desktops, Laptops, Tablets, and Mobile phones.
- **Performance Optimized:** Uses modern web practices and image formats (WebP).

## Technologies Used

- HTML5 (Semantic Structure)
- CSS3 (Flexbox, CSS Grid, Custom Properties)
- JavaScript (Vanilla JS for mobile nav & smooth scrolling)

## Project Structure

- `index.html`: Main file with semantic markup.
- `css/general.css`: Base styles, typography, and resets.
- `css/style.css`: Specific component and layout styles.
- `css/queries.css`: Media queries for responsive design.
- `js/script.js`: Interactive elements.
- `img/`: Highly optimized assets.

## Getting Started

To view the project locally:

1. Clone this repository.
2. Open `index.html` in your web browser.
   _(Alternatively, use an extension like Live Server in VS Code for a better development experience)._
